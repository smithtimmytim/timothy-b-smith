Easily add your Hello Bar to your WordPress blog.

== Description ==

The WP Hello Bar allows users to add the numeric codes from their HelloBar instance.


**Important Links:**

* [HelloBar.com](http://www.hellobar.com/)
* [Get Satisfaction support forum](http://www.getsatisfaction.com/hellobar)


== Installation ==
1. Upload the `wp-hello-bar` folder and all its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Copy the two numbers in your code snippet from [HelloBar.com](http://www.hellobar.com/) into the labeled areas.
